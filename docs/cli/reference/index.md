# `usage`

**Usage**: `usage [--usage-spec] [COMPLETIONS] <SUBCOMMAND>`

**Version**: 2.0.4

- **Usage**: `usage [--usage-spec] [COMPLETIONS] <SUBCOMMAND>`

## Arguments

### `[COMPLETIONS]`

Outputs completions for the specified shell for completing the `usage` CLI itself

## Flags

### `--usage-spec`

Outputs a `usage.kdl` spec for this CLI itself
