# CLI

## Installation

### [mise-en-place](https://mise.jdx.dev)

```sh
mise use -g usage
```

### Cargo

```sh
cargo install usage-cli
```

You can also use [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall) to install quicker:

```sh
cargo binstall usage-cli
```

### Homebrew

```sh
brew install usage
```
